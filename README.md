# Digital Warranty Vault

**Technologies:** Java, Spring Boot, MySQL  

Digital Warranty Vault is a web-based application that helps users securely store, manage, and track warranties of their purchased products. It allows uploading receipts, categorizing products, and sending automated reminders before warranty expiration.

## Features
- Upload product bills and warranty documents.
- Track warranty status and receive automated expiry reminders via email/SMS.
- Organize products by category, brand, or purchase date.
- Dashboard displaying product summary and upcoming expirations.
- Secure storage of documents with file path references in the database.
